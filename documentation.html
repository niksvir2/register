Документация системы аутентификации

О проекте

Система аутентификации с возможностью:

  • Регистрации новых пользователей
  • Входа в систему
  • Личного кабинета
  • Проверки авторизации
  • Безопасного хранения паролей (хеширование)

Структура проекта

  • index.html - Главная страница с формами входа/регистрации
  • login.php - Обработчик входа
  • register.php - Обработчик регистрации
  • dashboard.php - Личный кабинет
  • config.php - Конфигурация и "фейковая" БД
  • check_auth.php - Проверка статуса авторизации
  • logout.php - Выход из системы
  • style.css - Стили для всех страниц

Особенности реализации

  • Использование сессий для хранения состояния авторизации
  • Password hashing с помощью password_hash()
  • Валидация входных данных
  • Защита от XSS с помощью htmlspecialchars()
  • AJAX проверка авторизации при загрузке
  • Адаптивный дизайн

Ограничения

  • Используется "фейковая" БД в массиве (для реального использования требуется подключение к SQL)
  • Нет восстановления пароля
  • Нет подтверждения email
  • Нет защиты от CSRF-атак
